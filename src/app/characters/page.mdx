# Characters

Characters found in the Mass Effect universe. {{ className: 'lead' }}

## The character model

The character model contains all the information about a character, such as their name, species, gender and more.

### Properties

<Properties>
  <Property name="id" type="number">
    Unique ID for the character.
  </Property>
  <Property name="name" type="string">
    The character's name.
  </Property>
  <Property name="species" type="object">
    The species object for this character. Contains `id`, `name` and `url`
    properties.
  </Property>
  <Property name="gender" type="object">
    The gender object for this character. Contains `id`, `name` and `url`
    properties.
  </Property>
  <Property name="class" type="string">
    Description of the character's class e.g. job, role, rank, position or any
    other label that describes what the character primarily does.
  </Property>
</Properties>

## List all characters {{ tag: 'GET', label: '/api/v1/characters' }}

<Row>
  <Col>

    This endpoint returns a paginated list of all characters. By default, a maximum of ten results are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="limit" type="integer">
        Limit the number of characters returned. Defaults to 10 if not specified.
      </Property>
      <Property name="offset" type="integer">
        Offset the results by skipping the specified number of characters. Defaults to 0 if not specified.
      </Property>
    </Properties>

    <div className="not-prose">
      <Button
