# Quickstart

This guide will get you set up and ready to use ME-API.

<TryItOut />

## Overview

No authentication is required to access ME-API. All resources are open and fully available.

The API is versioned, and the current version is `v1`. The base URL for all requests is `https://me-api.njwon4.workers.dev/api/v1`.

Since this is a REST API, you can use your preferred language and HTTP client to make requests.

A good old cURL or fetch request will also work just fine:

- [cURL](https://curl.se/)
- [fetch](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)

## Making your first API request

After picking your preferred language and HTTP client, you are ready to make your first call to ME-API. Below, you can see how to send a GET request to the `characters` endpoint to get a list of characters.

<CodeGroup tag="GET" label="/v1/characters">

```bash {{ title: 'cURL' }}
curl -G https://me-api.njwon4.workers.dev/api/v1/characters
```

```js {{ title: 'JavaScript (fetch)' }}
fetch('https://me-api.njwon4.workers.dev/api/v1/characters')
  .then((response) => {
    return response.json()
  })
  .then((data) => {
    console.log(data)
  })
  .catch((error) => {
    console.error(error)
  })
```

```js {{ title: 'JavaScript (axios)' }}
import axios from 'axios'

axios
  .get('https://me-api.njwon4.workers.dev/api/v1/characters')
  .then((response) => {
    const data = response.data
    console.log(data)
  })
  .catch((error) => {
    console.error(error)
  })
```

```python
import requests

response = requests.get('https://me-api.njwon4.workers.dev/api/v1/characters')
data = response.json()
print(data)
```
