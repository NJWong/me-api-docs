# Genders

Species found in the Mass Effect universe. {{ className: 'lead' }}

## The gender model

The species model is very simple.

### Properties

<Properties>
  <Property name="id" type="number">
    Unique ID for the species.
  </Property>
  <Property name="name" type="string">
    The species name.
  </Property>
</Properties>

## List all species {{ tag: 'GET', label: '/api/v1/species' }}

<Row>
  <Col>

    This endpoint returns a paginated list of all species. By default, a maximum of ten results are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="limit" type="integer">
        Limit the number of species returned. Defaults to 10 if not specified.
      </Property>
      <Property name="offset" type="integer">
        Offset the results by skipping the specified number of species. Defaults to 0 if not specified.
      </Property>
    </Properties>

    <div className="not-prose">
      <Button
        href="/pagination"
        variant="text"
        arrow="right"
        children="Read the docs for pagination"
      />
    </div>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/api/v1/species">

    ```bash {{ title: 'cURL' }}
    curl -G https://me-api.njwong.workers.dev/api/v1/species -d limit=10
    ```

    ```js {{ title: 'JavaScript (fetch)' }}
    const response = await fetch('https://me-api.njwong.workers.dev/api/v1/species?limit=10')
    ```

## Retrieve a character {{ tag: 'GET', label: '/api/v1/characters/:id' }}

<Row>
  <Col>

    This endpoint allows you to retrieve information for a specific gender.

    ### URL parameters

    <Properties>
      <Property name="id">
        The unique ID of the gender to retrieve.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
